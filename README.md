# 🧠 Address Line Mapping Calculator

Memory Mapper Calculator adalah aplikasi Flutter yang membantu melakukan perhitungan dan visualisasi alokasi memori secara sekuensial untuk sistem mikroprosesor seperti AT89C51 (64KB address space).

## 🚀 Features

- ✅ Tambah komponen memori (RAM, ROM, I/O)
- ✅ Perhitungan otomatis start & end address
- ✅ Visual memory allocation progress (0 - 64KB)
- ✅ Toggle tampilan Hexadecimal / Decimal
- ✅ Drag & drop reorder memory block
- ✅ Save & Load project (local storage)
- ✅ Export ke Excel
- ✅ Recent files
- ✅ Delete file
- ✅ Clean & modern UI

---

## 🧮 Memory Rules

- Total address space: **0000H - FFFFH (64KB)**
- Sistem akan menolak alokasi jika melebihi 64KB
- Mapping dihitung secara sekuensial dari address 0
